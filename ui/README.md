# PharmaWatch
Drug risk intelligence platform. Enter a drug name and basic profile to get risk scores, side effect breakdowns, research digest, and a Q&A agent.

Built on FDA FAERS, PubMed, Reddit, LightGBM, ChromaDB, LangChain, and Groq.
